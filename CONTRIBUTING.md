# 贡献说明

## 开发流程

1. 基于 `master` 创建修改。
2. 优先编辑 `.ts`、`.scss` 和 `.wxml` 源文件。
3. 运行构建：

```bash
npm run build:mp
```

4. 运行测试：

```bash
npm test
```

5. 提交源文件和生成文件。

## 提交前确认

- 页面能在微信开发者工具中正常编译。
- 发票金额、税额、价税合计计算正确。
- 企业预设选择和手动输入都可用。
- 保存图片不会遮挡或裁切票面主要内容。

## 文档维护

如果修改了使用方式、构建方式、企业库来源或导出能力，请同步更新 `README.md` 和 `docs/DEVELOPMENT.md`。
